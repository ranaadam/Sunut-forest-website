# Sunut Forest Website

Sunut Forest is a university team project inspired by Sunut Forest in Sudan. It is an e-commerce website for gardening tools, ornamental plants, and tree saplings.

## Features
- Product categories
- Featured products
- Latest products
- Responsive layout
- Hover effects and clean UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Team Project
Developed collaboratively as part of a university project.

## Inspiration
Named after Sunut Forest in Sudan.
